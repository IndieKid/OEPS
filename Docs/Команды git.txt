Скачать git:
https://git-scm.com/download/

Репозиторий:
https://github.com/IndieKid/OEPS

Информация о команде:
git команда -h

Клонировать репозиторий в директорию:
git clone репозиторий <директория>

Посмотреть статус репозитория:
git status

Добавить конкретные файлы для синхронизации с репозиторием:
git add файл1 файл2 ...

Добавить все файлы для синхронизации с репозиторием:
git add *

Закомментировать и подготовить к обновлению в репозитории все измененные и добавленные файлы:
git commit -a -m "комментарий"

Загрузить изменения в репозиторий
git push

Скачать измения из репозитория
git pull

Убрать файл из списка синхронизации:
git update-index --no-assume-unchanged имя_файла
